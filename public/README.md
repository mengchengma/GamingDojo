# Logo files

Drop your transparent logo files in **this folder** with these exact filenames:

| File                  | What it is                          | Used in                              |
|-----------------------|-------------------------------------|---------------------------------------|
| `logo-shield.png`     | The "GD" shield mark (transparent)  | Nav, footer, brand-mark section       |
| `logo-wordmark.png`   | "GAMING DOJO" wordmark (transparent)| Nav, footer                           |

**Format:** PNG with transparent background works best. SVG is even better if you have it (just save it as `logo-shield.png` / `logo-wordmark.png` — the `<img>` tag accepts SVGs fine).

**Recommended sizes:**
- Shield: at least 256×256 px (renders at up to 176px in the brand-mark section)
- Wordmark: at least 480px wide

**If the files aren't present**, the site falls back to a stylized SVG version (lightning-bolt GD mark + text wordmark) using the active theme's brand color (cream on dark, navy on light). Everything still works — just less polished than your real artwork.
